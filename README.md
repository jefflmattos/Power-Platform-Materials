# Power Platform Learning Repository

Welcome to the **Power Platform Materials Repository**! This repository is dedicated to gathering and sharing learning materials for the Microsoft Power Platform, including PDFs, tutorials, useful links, and other resources to help you (and me) get started and master Power Platform tools.

## Contents of this Repository

This repository (will) contains various resources to assist in your learning journey with Power Platform. The materials are organized by their subject:

- **Architecture**: Based on best practices for design in development, configuration, integration, infrastructure, security, licensing, storage, and etc.


## Contributing

Contributions to this repository are welcome! If you have found a great tutorial, article, or resource that should be shared, feel free to submit a pull request. Please make sure the content is relevant.
